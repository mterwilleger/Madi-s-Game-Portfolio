SandBattle

My SandBattle game came from a tutorial from Zenva Academy found here. The base game spawned enemies with a loop and timer to make sure it wasn’t constantly spawning them. I used TileMap to create the map with a treasure hunt theme. As you go through the game you are collecting points, finding treasure, and destroying enemies. The game uses RPC’s to keep player and enemy health up to date. When a player reaches the “X”, it calls an RPC on everyone’s computer to end the game.

Thanksgiving

The base of this game came from a tutorial from Zenva Academy that can be found here. In developing this game, I implemented a leaderboard to track the time it took for players to collect all the objects. The game, made around Thanksgiving, reflects a focus on food which I thought was fitting. Each collected object contributes to the player's growth, which allows you to gather the remaining items faster. What sets this game apart is the player authentication system I created using Unity and Microsoft's PlayFab. PlayFab handles player sign-in and authentication, utilizing this information to store their time/score and rank them based on the fastest time achieved. Additionally, only the best time is saved for each player.
